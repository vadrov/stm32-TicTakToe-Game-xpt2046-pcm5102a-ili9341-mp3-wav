# stm32f4 Tic-Tak-Toe Demo Game

 #для stm32f4 (stm32f401ccu6) с touch XPT2046
 #Крестики-нолики
 
 С ИИ на основе алгоритма Минимакс (Minimax)
 Со звуком и эффектами (программные декодеры mp3 и wav)
